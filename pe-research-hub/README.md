# PE Research Hub

Read CHARTER.md before any work. It is the source of truth for goals, rules, and scope.
Read CONTEXT.md for company and supervisor context.

Standing rules (non-negotiable):
- Every number gets tagged SOURCED / ESTIMATE / UNVERIFIED. Never launder unverified data into clean-looking output.
- Frame everything as consolidating the team's research, never as solo work.
- Layered format: answer first, data second, appendix last. Matt has 2 minutes.
- Do the simplest thing that works. Do not add anything beyond what the current task requires. No new features, no extra tabs, no speculative sections.
- Phase 1 scope is exactly three artifacts (see PHASE1_PROMPT.md). Do not start Phase 2 work without being asked.
- Raw intern files live in raw/. Finished artifacts go to outputs/. Never edit files in raw/.
- Prose voice: short sentences, no em dashes, no hype vocabulary.
